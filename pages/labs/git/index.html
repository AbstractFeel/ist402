Let's start contributing to an open source project!

Let's start contributing to our first open source project! I've left a few issues in my own projects so that we can see the full project development life cycle at play here. I'm providing links below so your job is to fix one of these things and issue a pull request just like we did in the practice.

EVERYONE (me included) start out getting into a development community by updating documentation, making sure acronyms made sense and fixing links and typos. The best way to get into a software community is to fix things around the edges and ensure that you can effectively communicate with people in the project. It also helps prove they are human.
