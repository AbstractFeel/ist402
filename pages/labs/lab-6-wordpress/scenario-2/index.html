Lab Scenario 2 - Scope to nowhere

  • You won a contract from the Alaska state government to build a website to market: Alaska’s ”Bridge to nowhere” (look it up)
  • This site was originally built in Grav because you thought it was a one off, but now the client is mad because they heard a word other than “WordPress” (snark but happens ) and are instantly concerned about having to support two different applications .
  • You need to build a website and QUICK to convince them to stay with you, either as a new WordPress site or get them to use the existing one. This website needs turned around in 1 week (shocking ).
  • Pick a theme for a brochure style website and show them what a site in WordPress would feel like relative to Grav

Deliverable

  • Create a WordPress site on reclaim hosting
  • Select plugins to improve SEO, help ensure accessibility, and simplify content editing beyond the default editor (enable classic editor )
  • Pick a nice theme and customize it with a few links to make the site look like a real brochure style website (typical parks and rec site has History, News, Hours, Location, Reviews, etc).
  • Modify the homepage , be as snarky as you like, this is a real bridge project so meme it up. Under History make a Pros and Cons list of WordPress vs Grav . Make sure to list at least 3 advantages and disadvantages of each platform (links can help prove these).
  • Record a video demonstrating some of the similarities and differences between Grav and WordPress in order to allay their concern

Solving the scenario

  • The client is in favor of WordPress because of other projects. You like making money but want to cover your butt. Make arguments for why you agree they should migrate from Grav to WordPress . After all, this is a government contract and once your in maybe you can get more by making them happy.

    OR

  • You sell Grav over WordPres s; if you are going to get paid, might as well do it based on work you’ve already done rather then need to scrap it all and build them a new website. This might seem more difficult but remember.. Governments LOVE security and WP is hacked.. a lot. Play on that fear relative to how Grav works.

    Note: You don’t have to build a new Grav site, you can point to features in the existing one you made but you are welcome to make a new one if you so desire
